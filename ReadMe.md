# このソフトウェアについて

PythonでINIファイル読込する。

# 開発環境

* [Python 3.4.4](https://www.python.org/downloads/release/python-344/)
    * [configparser](http://docs.python.jp/3/library/configparser.html)

# 実行

```dosbatch
python IniRead.py
```

# 結果

INIファイルを読み込んでコンソールに表示する。
